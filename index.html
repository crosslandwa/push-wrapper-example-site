Keyboard/mouse control
  • play samples by clicking the buttons above
  • play samples by pressing your q-w-e-r-t-y-u-i keys
Push control
  • play samples using the grid. One column in the grid plays each sample
  • pads higher up the grid raise the sample filter frequency
  • press the pads harder to play samples louder
  • change sample pitch using the encoders
  • modulate pitch of all samples using the touchstrip
  • turn on/off note repeat for each sample with the select buttons above the grid
  • hold down a select button and press the time division buttons to change note repeat interval
  • change the BPM with the tempo knob

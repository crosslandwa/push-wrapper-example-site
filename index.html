Keyboard/mouse control
  • Play samples with the q-w-e-r-t-y-u-i keys
  • Select sequences with the 1-2-3-4-5-6-7-8 keys
  • Use the arm-play-delete buttons to record and play sequences in real time
Push control
  • Play samples using the Push's velocity sensitive pads - one column in the grid plays each sample
  • Pads higher up the grid raise the sample filter frequency
  • Press the pads harder to play samples louder
  • Change sample pitch using the encoders and modulate using the touchstrip
  • Turn on/off note repeat for each sample with the Select buttons above the grid
  • Change the note repeat interval with the Select + Time Division buttons
  • Select sequences with the row of Channel Select buttons below the Push LCD
  • Use the Record button to arm sequences - recording starts when you start playing samples
  • Use the Play button to start and stop recorded sequences
  • Use the Delete + Channel Select buttons to delete recorded sequences
Quantisation
  • With the metronome running sample recording/playback is quantised to the metronome's BPM
  • Turn the metronome on/off with the Push's Metronome button
  • Mute the metronome (but leave it running) by pressing the Push's Mute + Metronome buttons
  • Use Push's Tap Tempo button to update the metronome's BPM
  • Alternatively, change the BPM with the the Push's Tempo knob
  • Change the Metronomes accent (number of beats) with the Push's Shift button + Tempo knob
